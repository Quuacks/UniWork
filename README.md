# C Coursework Project Collection

This repository collects a broad set of small programming tasks and university mini-projects completed across multiple topics and weeks.  
It represents practical progress from basic problem solving to more structured program design, testing, and optimization.

## What this collection contains

- **Sequential assignment tasks (`uzd`)** that gradually increase in difficulty and scope.
- **Programming Paradigms labs (`PP laboras`)** emphasizing different ways of structuring and reasoning about code.
- **Mixed technical exercises** involving core C/C++ fundamentals, algorithmic thinking, data handling, and implementation discipline.

## General focus areas

- Writing clear, functional solutions for small but diverse programming problems.
- Applying key concepts such as conditionals, loops, functions, arrays/containers, and file/input handling.
- Improving code quality through iterative tasks, refactoring habits, and debugging practice.
- Building confidence with compilation, execution workflows, and project organization.

## Learning outcomes reflected by these tasks

- Stronger understanding of foundational programming principles.
- Better ability to break problems into smaller, implementable steps.
- Increased consistency in writing maintainable and testable code.
- Steady growth from isolated exercises toward more complete and structured project work.
